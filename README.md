# Dataset Descriptive Statistics App

This is a static web application that lets a user upload a CSV or TSV dataset and calculate descriptive statistics directly in the browser.

## Features

- Upload `.csv` or `.tsv` files
- Detect numeric columns automatically
- Show dataset-level metrics
- Display a summary table for numeric columns
- Inspect detailed descriptive statistics for a selected column
- Keep data local to the browser, which makes it suitable for GitHub Pages

## GitHub Pages deployment

1. Push these files to a GitHub repository.
2. In the repository settings, open **Pages**.
3. Set the deployment source to the branch that contains these files.
4. Use the root folder as the published site.

Because the app is fully static and runs entirely in the browser, no server setup is required.
